Five steps from job definition to final decision. Built so you and your team run the same process every time.

When to use this
⊕
Opening a new role
Build the job description before you post anything. Know exactly who you're looking for before you start reading resumes.
◎
Screening and interviewing candidates
Run a consistent phone filter and interview so you're comparing candidates on the same criteria, not gut feel.
⇄
Comparing multiple candidates
Add multiple candidates and see a side-by-side comparison of scores and decisions before making the call.
How it works
1 ·Job setup - Define the role, responsibilities, must-haves, and salary range.
2 ·Phone filter - Six quick questions to screen before investing in a full interview.
3 ·Interview - Seven scored sections covering experience, fit, and problem solving.
4 ·References - Two reference checks with structured questions.
5 ·Decision - Hire, hold, or pass. Offer checklist and onboarding prep.
Before you begin
→Complete Step 1 (job setup) before you post the role or screen anyone.
→Add each candidate using the "+ Add" button at the top. Steps 2-5 save per candidate.
→Use the Compare tab at the end to see all candidates side by side.
